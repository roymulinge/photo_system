# Photo App 

A simple, Instagram-inspired photo sharing web application built with Django.  
The project focuses on core social features, clean UI design, and learning how full-stack components connect together.

---

## Overview

Photo App allows users to:
- Create accounts and manage profiles
- Upload and view photos in a gallery
- Like photos
- View photo details
- Explore content in a clean, modern layout

The project emphasizes **backend fundamentals**, **Django architecture**, and **Tailwind CSS styling**.

---

## Features

- User authentication (register, login, logout)
- User profiles with bio and profile picture
- Photo upload with title, description, and tags
- Photo gallery with hover effects
- Like and unlike photos
- Photo detail view
- Responsive design using Tailwind CSS

---

## Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, Tailwind CSS
- **Database:** SQLite
- **Authentication:** Django Auth System
- **Styling:** Tailwind CDN
